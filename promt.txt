Создай страницу /admin для подтверждения достижений.

Требования:

1. Доступ только для пользователей с is_admin = true в profiles.

2. Страница /admin показывает список всех достижений со статусом 'pending':
   - Категория, название, описание, год, доказательство (фото/ссылка)
   - Кнопки "Подтвердить" и "Отклонить"

3. При нажатии "Отклонить" — модалка с полем "Причина отклонения" (обязательное).

4. После действия:
   - Обновить статус в таблице achievements (status = 'verified' или 'rejected')
   - Если rejected — заполнить rejection_reason
   - Убрать достижение из списка pending (или обновить статус на странице)

5. В таблице achievements уже есть поля:
   - status (pending / verified / rejected)
   - rejection_reason (текст)

6. Используй компоненты @primer/react для интерфейса.

7. Добавь ссылку на /admin в Header (только для админа).

После выполнения проверь:
- Войди под админом (is_admin = true)
- Увидел список достижений на проверке
- Подтвердил одно, отклонил другое
- Статусы обновились в БД
