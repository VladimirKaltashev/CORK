# CORK — Clown or King

## TL;DR

**Социальная сеть с приговором.** Каждое достижение — ставка: ты король или клоун. Решает не админ, не алгоритм, а толпа. Из лайков убраны сердечки — остались только 👑 **корона** и 🤡 **клоун**.

Реакции **ограничены и стоят денег**: 10 голосов в неделю, корона стоит 1 голос, клоун — 2. Дешёвой токсичности нет, дешёвых лайков тоже. Каждая реакция — решение.

CORK — это **рынок репутации для школьников**: твои победы и публичная цена каждой из них.

---

## Проблема

У школьника два параллельных дефицита:

1. **Дефицит признания за реальные победы.** Дипломы лежат в папке у мамы, грамоты — в шкафу. Соцсети награждают сторис, а не победы в олимпиаде.
2. **Дефицит честной обратной связи.** Лайки в инсте бесплатны и поэтому ничего не значат. Похвала родителей слепая. Друзья тактично молчат.

Существующие соцсети не закрывают ни один из них — там нет ни верификации достижений, ни обратной связи с ценой.

---

## Решение

### Достижения = твой контент
Олимпиада, проект, грамота, спортивная победа, выпущенная игра. Любая категория. Проходит модерацию админа: верифицированное достижение получает плашку ✅ и попадает в общую ленту.

### Реакции = твоя цена
На любое достижение в ленте можно поставить:
- 👑 **корона** — «это уровень», стоит **1 голос**.
- 🤡 **клоун** — «это смешно», стоит **2 голоса**.

Никакой защищённой зоны. Реальные дипломы тоже можно заклоунить — но это **дорого** (см. ниже), поэтому случайно не прилетает.

---

## Реакции — экономика дефицита

| Параметр | Значение |
|---|---|
| Бюджет | **10 голосов в неделю** |
| Цена короны 👑 | **1 голос** |
| Цена клоуна 🤡 | **2 голоса** |
| Сброс | Понедельник 00:00 |
| Остаток | Сгорает, не копится |

**Почему так:**

- **Дефицит даёт вес.** Лайк бесплатен — поэтому ничего не значит. Корона стоит «1 из 10» — поэтому это уже мнение.
- **Асимметрия 1/2 страхует от травли.** Клоун дороже короны. Чтобы заклоунить, ты вкладываешь вдвое больше, чем чтобы похвалить. Низкоусилийной злобы не существует — клоун всегда инвестиция.
- **Сброс по понедельникам = retention-loop.** Свежие звёзды = повод зайти. Бесплатный возврат пользователя.
- **Сгорание = urgency.** Не потратил — потерял.

Сверху над фидом всегда виден виджет: `у тебя осталось 7 голосов · сброс через 3 дня`.

---

## Профиль = карма

```
👑 247   🤡 38   ratio 6.5
```

Соотношение крон к клоунам — это твой публичный статус. Не «100 лайков», а **«247 к 38»**. Сравнимо, осмысленно, спорно.

На профиле — список твоих достижений с реакциями на каждое.

---

## Кому это нужно

### Основная аудитория — школьник 7–11 класса
Хочет быть оценённым ровесниками, а не родителями. Готов к честной обратной связи и к соревнованию. Уже живёт в ratio-культуре твиттера/тиктока.

### Вторичные роли
- **Друг / одноклассник** — главный участник арены. Тратит звёзды, спорит реакциями.
- **Приёмная комиссия / стажировка** — открывает один URL, видит верифицированные достижения. Клоуны для них — побочный шум, верифицированный диплом всё равно остаётся верифицированным.

## Ключевые фичи MVP

| Фича | Что даёт юзеру |
|---|---|
| 👑 / 🤡 реакции с бюджетом | Каждая реакция — решение, а не рефлекс. Антирейтинг с весом, без травли. |
| **Достижения** (с верификацией) | Публичный CV. Можно скинуть приёмке. |
| **Лента** (Все / Друзья + категории) | Видишь, чем живут ровесники и друзья. |
| **Скор на профиле** | 👑 / 🤡 / ratio. Карма видна. |
| **Друзья и поиск** | Найти своих, добавить, видеть их в ленте «Друзья». |
| **Bio + контакты** | Telegram / GitHub / своя ссылка. Аватар. Публичная страница. |

## Пути пользователя

### 1. Первый вход

```
Регистрация  →  Автосоздание профиля  →  10 стартовых звёзд  →  Coachmark-тур
```

Тур объясняет: реакции стоят, у тебя бюджет, корона = 1, клоун = 2.

### 2. Первая реакция

```
Открыл фид  →  Увидел достижение  →  Жмёт 👑 (−1)   ИЛИ   жмёт 🤡 (−2)
                                                ↓
                                      Виджет бюджета обновляется:
                                      «осталось 9 голосов»
```

### 3. Первое достижение

```
Аватар  →  Мой профиль  →  «Добавить достижение»  →  Заполнить форму
                                                       (категория, год, фото/ссылка)
                                                                 ↓
                                                          Статус: pending
                                                                 ↓
                                                       Модератор проверяет
                                                                 ↓
                                                       verified → видно в ленте всех
                                                                 ↓
                                                       Толпа начинает голосовать
```

### 4. Возврат «постоянного» пользователя

```
Понедельник  →  «У тебя 10 свежих звёзд»  →  Открыть ленту «Друзья»
                            ↓
                  Потратил 4 короны и 1 клоуна за вечер
                            ↓
                  Осталось 4 голоса на остальную неделю
```

---

## Зачем именно так

### Почему не «ещё одна соцсеть»
- В обычных соцсетях реакция бесплатна → шум. Здесь реакция стоит → сигнал.
- Контент сфокусирован: только достижения. Нет бесконечной ленты про «что я ем».

### Почему не Behance / GitHub / LinkedIn
- LinkedIn — для взрослых, школьнику там не место.
- Behance / GitHub — только одна вертикаль.
- Здесь — **любая** категория + игровая механика, а не CV-сервис.

### Почему антирейтинг — не зло
1. **Бюджет.** У тебя 10 голосов в неделю — заклоунить всё подряд физически нельзя.
2. **Цена клоуна = 2 голоса.** Каждый клоун стоит вдвое больше, чем корона. Дешёвой злобы нет — клоун всегда инвестиция.

В сумме: антирейтинг превращается из «инструмента травли» в **рынок мнений с ценой ошибки**.

---

## Метрики, по которым меряем успех

- **DAU школьников 7–11 класса**
- **Доля недельного бюджета, которая тратится** (= вовлечённость). Цель: >70%.
- **Среднее число достижений на активный профиль**
- **Возврат по понедельникам** (= работает ли retention-loop сброса звёзд)
- **Среднее ratio (👑/🤡) среди активных юзеров**

## Дорожная карта (v2 и дальше)

Ниже — фичи, которые отложены из MVP и появятся в следующих версиях.

- 🎯 **Цели / вишлист** — публичные ставки на будущее: «Хочу выиграть олимпиаду в 2026». Толпа реагирует короной/клоуном. При выполнении цели → конвертируется в достижение, проголосовавшие получают результат своего прогноза.
- 🏆 **Челленджи** — недельные и месячные темы. «Книги недели», «Худший рисунок мышью», «Прогресс по своей цели». Бейджи «Король недели», «Клоун месяца».
- 🎭 **Бейджи** — постоянные награды за победы в челленджах.
- 🔮 **Оракул** — счётчик верных прогнозов на чужие цели. Игра в предсказателя.
- 💬 **Комментарии и чаты** — обсуждение под достижениями, личка.
- 📝 **Свободные посты** — флексы, мемы, мнения вне формата «достижение».
- 🗳️ **Юзер-предложенные челленджи** — комьюнити голосует за тему недели.
- 👥 **Группы / классы / команды** — общая лента и общие челленджи.
- 🏅 **Лидерборды** — по школам, регионам, категориям.
- 📤 **Экспорт профиля в PDF**
- 🔔 **Уведомления**
- 🌐 **Публичная страница без логина** — `cork.app/u/name`
- ⭐ **Премиум-бусты** — +5 звёзд за активность или достижение
